About ProfileGuard

ProfileGuard is a tool designed to detect fake social media profiles using a hybrid SVM-Neural Network model. By analyzing features like follower count, post frequency, and privacy settings, we help you identify suspicious accounts with ease.

Built with Flask and machine learning, this project leverages datasets from Kaggle to provide accurate predictions. Stay safe online!
